# Design Brief: CatFinity

**Purpose:** Tutorial platform for learning (∞, 1)-categories theory. Scholarly, distraction-free, math-forward. **Tone:** Editorial academic refined. **Palette:** Deep navy primary (0.32 0.08 258), warm cream/off-white (0.98 0 0), amber accent (0.62 0.16 58), warm greys for secondary. **Typography:** Fraunces (serif display, headings), General Sans (body, math content), Geist Mono (inline code). **Shape:** 8px radius, refined shadows. **Differentiation:** Serif serif pairing for academic gravitas; KaTeX formulas elevated visually; card-based lesson grid; strong dark mode.

| Mode       | Background        | Card              | Text             | Primary          |
|------------|-------------------|-------------------|------------------|------------------|
| Light      | 0.98 0 0 (cream)  | 0.995 0 0 (white) | 0.20 0 0 (navy)  | 0.32 0.08 258    |
| Dark       | 0.14 0 0 (char)   | 0.17 0 0 (slate)  | 0.93 0 0 (off)   | 0.75 0.08 258    |

**Structural Zones:** Header (nav + theme toggle, navy primary bg light / dark overlay), content grid (card-based lessons 2–3 cols), footer (minimal, muted). **Spacing:** 2rem max-width margins, 1.5rem card padding, 1rem section gaps. **Elevation:** Cards use subtle shadow (0 2px 8px), hover elevation (0 8px 24px). **Motion:** Smooth 0.3s cubic-bezier transitions on card hover, theme toggle. **Component Patterns:** Button hierarchy (primary navy, secondary muted), lesson cards with title/blurb/progress, KaTeX formula blocks inline. **Constraints:** No decorative gradients, no animations beyond hover/transitions, high contrast WCAG AA+. **Detail:** Serif pairing (Fraunces + General Sans) signals scholarly tone; amber accents highlight formulas and CTAs sparingly.
